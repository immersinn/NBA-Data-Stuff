Welcome to the nba_analytics wiki!

Due to data now posted on the http://stats.nba.com/ page (edit: this is currently "under construction" / unavailable due to updates being made), player and ball position data can now be processed and analyzed. The current state of affairs focuses on this main pipeline:

*Scrape the data from the NBA page *Make the data consumable for analytics purposes *Pre-process and clean the data (specifically the Play-by-play and Moments / Movements data) *Merge the PBP and Moments data with the purpose of aligning player and ball positions with game plays *Extract the ball transition network from each game (see http://journals.plos.org/plosone/article?id=10.1371/journal.pone.0047445)
